Experiment Lab


  1. Click on the link and complete the survey: https://www.surveymonkey.com/s/1314TEALSsurvey

  2. After you have submitted the survey, go here and download the BYOB template for this lab.

  3. Open the lab-3 template from your downloads folder. This is a prebuilt program that you are to modify. Take a moment to notice that there are 3 sprites on the stage. Each one of them responds to key presses, which are listed on the stage.

  4. Perform each modification to the program:

    1. Walk Sprite: Make the sprite move diagonally accross the screen
    2. Walk Sprite: Import new costumes of your choice to replace the current costumes
    3. Walk Sprite: Replace the 'change x', 'change y' blocks with the glide to block

    4. Draw Sprite: Add 'change color effect by _' block
    5. Draw Sprite: Change side_length and angle variable at the bottom of the screen
    6. Draw Sprite: Instead of adding 5 to the variables in the 'move _ steps' 'turn _ degrees', try adding a different number

    7. Squirrel Sprite: Identify the if block in the code
    8. Squirrel Sprite: What do you have to do to get that code to run?
    9. Squirrel Sprite: Within the if block, instead of changing the color effect, select a different effect from the dropdown to edit
    10. Squirrel Sprite: Play a sound within the if block

  5. After that, see what interesting modifications you can come up with. Today is all about experimentation, so try things out and see what happens! If you don't know what something does, play around with it to see if you can figure it out.

  6. At the end of class, please save your file as 'lab3-firstname-lastname' and send it to leaders.hs.cs.2013@gmail.com
